# 학부모 유저 플로우

## 대상

- 중3부터 고3까지 자녀를 둔 학부모
- 고교학점제, 수시, 생기부, 과목 선택, 학교생활 설계가 낯선 학부모
- 대치동식 고가 관리에는 부담을 느끼지만 자녀의 방향성은 확인하고 싶은 학부모

## 핵심 목적

> "내 아이가 고3까지 어떤 과목·활동·탐구 선택과 학교생활을 해야 하는지 한눈에 보고, 불안을 실행 계획으로 바꾼다."

## 1차 유입 흐름

1. 광고·커뮤니티·지인 추천을 통해 랜딩 페이지 진입
   - 맘카페
   - 네이버 카페
   - 당근마켓
   - 학교/지역 커뮤니티
   - 기존 사용자 추천
2. 문제 공감
   - "지금의 학교생활 선택이 고3 생기부로 이어진다"
   - "고3까지의 생기부 흐름을 먼저 보고 지금 선택한다"
   - 고교학점제, 과목 선택, 수행평가, 탐구 활동 부담 이해
3. 결과물 샘플 확인
   - 미래 생기부 미리보기
   - 강점/약점 요약
   - 하루/주간/월간/학기 액션
   - 고3까지의 학교생활 로드맵
4. 자녀와 함께 무료 미리보기 시작
   - 자녀 기본 정보 입력
   - 생기부·활동 자료 입력 동의
   - 개인정보 처리 안내 확인
5. 결과 확인
   - 자녀의 현재 위치
   - 가능한 진로 후보
   - 고3까지의 학교생활 방향
   - 이번 달/이번 학기 해야 할 액션
6. 공유·저장·후속 신청
   - 결과 링크 저장
   - PDF 또는 요약본 저장
   - 베타 신청 또는 사전예약
   - 출시 알림 신청

## 반복 사용 흐름

1. 월간 요약 확인
   - 자녀가 기록한 활동
   - 이번 달 실행한 액션
   - 진로 일관성 변화
   - 부족한 활동 영역
2. 다음 달 우선순위 확인
   - 과목 선택
   - 수행평가 주제
   - 독서·동아리·탐구 활동
   - 생기부와 연결되는 학교생활 우선순위
3. 결제 또는 구독 판단
   - 무료 결과물이 충분히 신뢰되는지 판단
   - 지속 갱신 가치 확인
   - 학기 단위 또는 월 단위 상품 검토

## 학부모 화면에서 필요한 주요 기능

- 자녀 결과 요약
- 하루/주간/월간/학기 액션 진행률
- 활동 갭 알림
- 월간 리포트
- 개인정보 동의·보유기간 안내
- 사전예약/출시 알림 신청
- 결제 의향 확인 설문
