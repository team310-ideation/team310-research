# 시장규모 TAM/SAM/SOM

## 핵심 방향

- 기획서에는 TAM, SAM, SOM 3단계 시장 규모 분석이 필요하다.
- 전체 입시 시장을 바로 매출 시장처럼 주장하지 않고, 실제 접근 가능한 시장과 초기 수익화 시장을 분리한다.

## 구조

| 구분 | 의미 | 미리 적용 방향 |
| --- | --- | --- |
| TAM | 전체 시장 | 중·고등학생 입시/진로/생기부 관리 전체 수요 |
| SAM | 유효 시장 | 고교학점제, 학생부종합전형, 고입/대입 전략에 관심 있는 학부모·학생 |
| SOM | 초기 수익화 시장 | 5월 MVP에서 AI 리포트/베타 테스트에 반응하는 중3~고1 학부모·학생 |

## 검증해야 할 질문

- [ ] 주요 타겟이 자신의 진로를 위해 월 3만 원을 결제할 수 있는가?
- [ ] 주요 타겟이 정말로 진로 설계를 원하는가, 아니면 부모만 문제를 느끼는가?
- [ ] 다수 학생 대상 합리적 가격 모델에서 차별점과 강점은 무엇인가?
- [ ] 중등과 고등 중 6개월 안에 어느 시장을 먼저 잡을 것인가?

## 시장 검증 원칙

- 유저 1명만으로는 PMF 검증이 불가능하다.
- 최소 10명 이상의 베타 테스터 풀을 확보한다.
- 다양한 세그먼트를 포함한다.
  - 소득: 중산층, 고소득, 저소득
  - 지역: 수도권, 비수도권
  - 진로: 특목고, 일반고, 예체능
- "중3 학부모는 위기감이 있는가?"는 이론보다 실제 유저풀 확보로 검증한다.

## 시장 변화 트렌드

- 고교학점제 도입으로 입시·생기부 관리 시장이 팽창 중이라는 가설이 있다.
- 시장은 "분석 후 대학 추천"에서 "3년 풀 관리"로 진화하고 있다.
- 풀세팅, 전형 풀관리, 심층 풀관리 등 장기 관리 패키지화가 관찰된다.
- 학생부종합전형은 회의 메모 기준 전체 대학 수시 44%와 연결되어 언급되었다. ❓ 공식 수치와 기준연도 확인 필요.

## 현재 시장에서 봐야 할 질문

- [ ] 고교학점제 변화가 실제 결제 의사로 이어지는가?
- [ ] 중3 → 고1 진입 시점이 가장 강한 구매 타이밍인가?
- [ ] 7월 방학 수행평가 준비 시즌이 더 좋은 진입 타이밍인가?
- [ ] 3년 풀 관리 시장의 가격대와 고객군은 어디까지인가?
