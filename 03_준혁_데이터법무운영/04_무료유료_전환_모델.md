# 무료→유료 전환 모델

## 한 줄 결론

미리(MIRI)는 처음부터 “유료 컨설팅”으로 팔기보다 **무료 학생부 진단 Lite → 일부 결과 잠금 → 3만 원대 전략 리포트 → 10만 원대 코치 패키지**가 자연스럽다. 무료 기능은 충분히 유용해야 하지만, “정확한 로드맵/면접 질문/전형 후보”는 유료로 잠가야 한다.

## 교육 서비스 전환 패턴

| 패턴 | 설명 | 사례형 힌트 | 미리 적용 |
|---|---|---|---|
| 무료 체험 | 일정 기간 또는 일부 기능 무료 | 콴다 프리미엄 7일 체험형 구조 | 1회 무료 진단 |
| 기능 제한 | 기본 기능 무료, 심화 기능 유료 | AI 문제풀이/해설 앱 | 강점 3개 무료, 로드맵 유료 |
| 결과 잠금 | 분석은 완료, 상세 결과는 결제 후 공개 | 리포트형 서비스 | 부족 요소 수만 무료 표시 |
| 상담 전환 | 무료 결과 후 상담/검수 판매 | 입시 컨설팅 | 컨설턴트 검수 옵션 |
| B2B 체험 | 학교/학원 무료 체험 후 기관 구매 | 클래스팅 AI 무료 체험 후 구매 | 학원 10명 무료 진단 |

## 미리 상품 구조

| 상품 | 가격 가설 | 제공 내용 | 목적 |
|---|---:|---|---|
| 무료 진단 Lite | 0원 | 섹션 감지, 강점 3개, 약점 3개, 분석 가능 항목 수 | 리드 확보 |
| 전략 리포트 | 29,000~39,000원 | 진단, 전공 적합도, 성장 로드맵, 면접 질문 | 첫 매출 |
| 코치 패키지 | 99,000~199,000원 | 학기별 계획, 탐구 주제, 답변 피드백, 컨설턴트 검수 | 객단가 상승 |
| 학원/B2B | 월 구독/건당 과금 | 학생별 진단, 리포트 일괄 생성, 상담 메모 | 반복 매출 |

## 무료에서 어디까지 보여줄까

### 보여줘야 하는 것

- 파일이 제대로 분석됐다는 확신
- 강점 3개
- 약점 3개
- 희망학과와 연결되는 근거 수
- 생성 가능한 면접 질문 수
- 빈틈 전형 후보 수

### 잠가야 하는 것

- 구체적인 로드맵
- 과목별 세특 소재
- 면접 질문 전체
- 전공 적합도 상세 근거
- 빈틈 전형 후보 리스트
- PDF/Markdown 다운로드

## 무료 진단 화면 예시

```text
분석 완료

감지된 학생부 항목: 42개
강점 후보: 3개
보완 필요 요소: 4개
희망학과 연결 근거: 7개
생성 가능한 면접 질문: 12개
검토 가능한 전형 후보: 5개

상세 리포트에서 확인할 수 있는 것:
- 전공 적합도
- 이번 학기 활동 로드맵
- 면접 예상 질문
- 학생부 근거별 보완 방향
```

## 결제 전환 포인트

### 1. 불안 해소

학부모는 “우리 아이가 늦었나?”를 알고 싶어 한다.

전환 문구:

```text
지금 학생부에서 가장 먼저 보완해야 할 3가지를 확인하세요.
```

### 2. 구체 행동

학생은 “그래서 뭘 해야 해?”를 알고 싶어 한다.

전환 문구:

```text
이번 학기 과목별로 세특에 남길 수 있는 활동 아이디어를 받아보세요.
```

### 3. 면접 대비

고3은 바로 쓸 결과물이 필요하다.

전환 문구:

```text
학생부 기반 예상 질문과 꼬리질문을 받아보세요.
```

## 가격 실험안

| 가격 | 기대 반응 | 리스크 |
|---:|---|---|
| 9,900원 | 첫 결제 장벽 낮음 | 컨설팅 신뢰가 낮아 보일 수 있음 |
| 29,000원 | 가볍지만 진지한 리포트 가격 | 결과 품질이 약하면 환불 리스크 |
| 39,000원 | 전략 리포트로 설득 가능 | 무료 대비 차이가 분명해야 함 |
| 99,000원 | 코치 패키지 가능 | 사람 검수 없으면 비싸 보일 수 있음 |

## 추천 초기 상품

### 무료

학생부 진단 Lite

### 첫 유료

`29,000원 전략 리포트`

구성:

- 학생부 진단
- 전공 적합도
- 이번 학기 로드맵 5개
- 면접 예상 질문 10개
- 위험 표현/보완 체크리스트

### 업셀

`99,000원 코치 패키지`

구성:

- 전략 리포트
- 탐구 주제 5개
- 면접 답변 피드백
- 컨설턴트 검수 1회

## 주의할 점

- “무료로 다 보여주고 PDF만 유료”는 전환이 약할 수 있다.
- “너무 조금 보여주기”는 신뢰를 못 만든다.
- 실제 결제 전환율은 공개 자료로 확정하기 어렵다. 반드시 fake-door나 사전예약으로 검증해야 한다.

## 멘토에게 물어볼 것

- ❓ 29,000원 전략 리포트가 학부모에게 가볍게 살 가격인지?
- ❓ 99,000원 이상은 사람 검수 없이는 어려운지?
- ❓ 무료 진단 결과를 얼마나 보여줘야 신뢰와 결제가 동시에 생기는지?
