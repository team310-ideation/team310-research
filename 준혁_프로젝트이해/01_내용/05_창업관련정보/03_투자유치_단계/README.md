## 정리 기준

- 정부지원사업과 VC 투자유치는 성격이 다르므로 별도 폴더로 관리한다.
- 단계는 `지원금/부트스트랩 → 프리시드 → 시드 → 프리A → 시리즈A → 시리즈B 이후` 순서로 본다.
- 미리(MIRI)는 현재 `지원금/부트스트랩 단계 → 프리시드 준비 단계`로 분류한다.
- 당장 투자유치보다 `문제 검증`, `MVP 데모`, `베타 사용자`, `반복 사용 지표` 확보가 우선이다.
