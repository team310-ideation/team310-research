# 학생 유저 플로우

## 대상

- 중3부터 고3까지의 학생
- 고교학점제, 과목 선택, 수행평가, 탐구 활동, 생기부 방향이 막막한 학생
- 현재 관심사는 있지만 학교생활의 선택들이 고3 생기부 서사로 어떻게 이어질지 그려지지 않는 학생

## 핵심 목적

> "AI와 함께 고3까지의 과목·활동·탐구 선택과 생기부에 연결되는 학교생활 전반을 설계한다."

## 1차 사용 흐름

1. 랜딩 페이지 진입
   - "고3까지 내 학교생활과 생기부는 어떻게 쌓일까?" 메시지 확인
   - 샘플 미래 생기부 미리보기 확인
2. 무료 미리보기 시작
   - 이름 또는 닉네임 입력
   - 학년, 학교 유형, 관심 진로, 현재 고민 입력
3. 기초 데이터 입력
   - 기존 생기부 또는 활동 자료 업로드
   - 자료가 없으면 관심사·활동 경험 중심으로 진행
4. AI 진로 인터뷰
   - 좋아하는 과목
   - 최근 몰입한 활동
   - 싫어하거나 어려운 것
   - 고등학교 생활에서 걱정되는 것
   - 관심 학과·직업 후보
5. 미래 생기부 미리보기 확인
   - 강점/약점 요약
   - 진로 후보
   - 고3까지의 학교생활 방향성
   - 과목·동아리·탐구·독서 추천
   - 하루/주간/월간/학기 액션
6. 첫 액션 선택
   - 이번 달 할 활동 1개 선택
   - 수행평가 주제, 독서, 동아리 활동, 탐구 주제 중 하나로 시작
7. 결과 저장 또는 공유
   - 결과 페이지 저장
   - 학부모에게 공유
   - 사전예약 또는 베타 신청

## 반복 사용 흐름

1. 활동 기록 추가
   - 수행평가 결과물
   - 발표 자료
   - 독서 기록
   - 동아리 활동
   - 교내 대회·프로젝트 결과
2. AI가 활동을 분류
   - 과목
   - 진로 키워드
   - 생기부 항목
   - 강점/약점
3. 리포트 갱신
   - 현재 생기부 방향 업데이트
   - 고3까지의 학교생활 로드맵과의 차이 확인
   - 다음 액션 추천
4. 학생이 다음 선택 실행
   - 과목 선택
   - 활동 선택
   - 탐구 주제 선택
   - 기록 보완
   - 생기부와 연결되는 학교생활 의사결정

## 학생 화면에서 필요한 주요 기능

- 미래 생기부 미리보기
- AI 진로 인터뷰
- 하루/주간/월간/학기 액션 체크리스트
- 활동 아카이빙
- 수행평가·탐구 주제 추천
- 과목 선택·학교생활 로드맵
- 진로 일관성/활동 갭 알림
- 학부모 공유 링크
