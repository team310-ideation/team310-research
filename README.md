## 3. AI 에이전트 협업 룰 (서로 다른 AI 써도 결과물이 호환되도록)

### 룰 1 — 모든 결과물은 마크다운 (.md)

PPT·노션·구글닥 직접 쓰지 말 것. `.md` 만 사용. AI 에이전트마다 출력 포맷이 다른데, 마크다운은 다 호환됨.

### 룰 2 — 모든 파일 첫 줄에 메타 헤더

```markdown
---
담당: 세민 / 세종 / 준혁
AI: GPT-5 / Claude Sonnet 4.6 / Gemini 2.5 Pro / Codex
작성일: 2026-04-28 14:30
주제: <한 줄>
신뢰도: 높음 / 중간 / 낮음 (멘토에게 확인 필요)
---
```

→ 이 헤더만 보면 다른 사람이 *"이거 어디서 온 정보인지, 검증된 건지"* 즉시 파악 가능.

### 룰 3 — 출처는 무조건 링크

AI가 답한 내용은 모두 링크 또는 검색 키워드 첨부. 출처 없으면 **신뢰도: 낮음** 으로 표기.

### 룰 4 — 본인이 모르는 건 "❓ 멘토에게 물어볼 것" 표시

조사하다가 막히면 답을 만들지 말고 `❓` 마크로 남기기. 마지막에 `00_취합/멘토_질문리스트.md` 로 모음.

### 룰 5 — 결론 1페이지 필수

각자 폴더 마지막에 `00_결론_1페이지.md` 작성. 멘토 미팅에 들어갈 핵심만.

## 5. Git 사용 (선택)

이 폴더를 GitHub 레포로 푸시하려면:

```bash
cd team310-research
git init
git add .
git commit -m "initial: 2차 멘토 미팅 준비 리서치 시작"

# GitHub에 새 private 레포 만든 후
git remote add origin https://github.com/team310/research.git
git branch -M main
git push -u origin main
```

→ 시간 없으면 그냥 카톡·슬랙으로 zip 공유해도 OK. 핵심은 **3명이 같은 폴더 구조를 갖고 같은 양식으로 쓰는 것**.
