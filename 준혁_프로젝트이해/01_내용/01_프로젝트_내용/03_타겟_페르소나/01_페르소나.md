# 학부모 페르소나 후보

## 목적

미리(MIRI)의 초기 타겟 페르소나는 학생이 아니라 **학부모**로 둔다.

학생은 실제 사용자이자 데이터 제공자지만, 초기 구매 의사결정과 문제 제기의 주체는 학부모다. 따라서 MVP와 인터뷰에서는 "어떤 학생인가"보다 "어떤 학부모가 어떤 불안을 느끼고, 어떤 결과물을 보면 결제하거나 공유할 수 있는가"를 먼저 검증한다.

## 정리 원칙

| 구분 | 정의 |
| --- | --- |
| 페르소나 주어 | 학부모 |
| 실제 사용자 | 중3~고3 학생 |
| 결제·공유·신뢰 판단 | 학부모 |
| 학생 정보의 역할 | 자녀 상황, 사용 맥락, 데이터 입력 맥락 |
| 검증 기준 | 학부모가 문제를 강하게 느끼는가, 결과물을 이해하는가, 비용을 낼 이유가 생기는가 |

## 후보 요약

| 후보 | 학부모 상황 | 자녀 상황 | 핵심 불안 | 미리의 제공 가치 | 우선순위 |
| --- | --- | --- | --- | --- | --- |
| A. 중3 진입 불안형 학부모 | 고등학교 진학을 앞두고 입시 구조를 처음 체감함 | 중3, 일반중, 진로·고교 선택 전 | 고교 선택과 고교학점제 이후 학교생활 설계를 어떻게 시작해야 할지 모름 | 고교 선택 기준, 3년 학교생활 로드맵, 첫 30일 액션 | 높음 |
| B. 고1 수시 포기 방지형 학부모 | 고1 첫 시험 이후 수시를 계속 가져가야 할지 흔들림 | 일반고 고1, 중위권~중상위권 | 생기부 방향을 놓쳐 이미 늦은 것 아닌지 불안함 | 과목·활동·탐구 공백 진단, 학기 단위 회복 전략 | 높음 |
| C. 비수도권 정보격차형 학부모 | 지역 내 입시 정보와 사례가 부족함 | 비수도권 중3~고2 | 수도권 학생만 아는 정보를 우리 아이만 모르는 것 같음 | 온라인 기반 개인화 로드맵, 지역 격차 보완용 판단 기준 | 중간~높음 |
| D. 특수진로 서사화형 학부모 | 자녀의 관심사는 분명하지만 입시 서사로 바꾸기 어려움 | 예체능·요리·디자인·국제기구 등 특수 관심사 | 좋아하는 일이 생기부 경쟁력으로 이어질 수 있는지 모르겠음 | 관심사를 과목·탐구·활동·독서로 번역 | 중간 |
| E. 특목·프리미엄 비교형 학부모 | 이미 정보 탐색량이 많고 더 나은 선택지를 찾음 | 특목고·자사고·상위권 일반고 고민 중 | 어떤 고교 유형이 자녀에게 유리한지 객관적으로 비교하기 어려움 | 고교 유형별 시뮬레이션, 선택 기준 정리 | 중간 |
| F. 월간 점검 니즈형 학부모 | 자녀가 잘하고 있는지 주기적으로 확인하고 싶음 | 중3~고3, 진로·활동 방향이 불명확함 | 지금의 선택이 맞는지, 다음에 무엇을 해야 하는지 계속 불안함 | 월간·학기 리포트, 다음 액션, 학교생활 진행 상황 요약 | 높음 |

## A. 중3 진입 불안형 학부모

| 항목 | 내용 |
| --- | --- |
| 예시 이름 | 박세민 학생 어머니 |
| 학부모 상황 | 중3 자녀의 고등학교 선택을 앞두고 고교학점제, 수시, 생기부를 한꺼번에 이해해야 하는 상황 |
| 자녀 상황 | 수도권 일반중 중3, 중상위권, 제과·미술·리더십 활동에 관심이 있음 |
| 핵심 감정 | "고등학교 들어가기 전에 방향을 잡아줘야 할 것 같은데, 어디서부터 시작해야 할지 모르겠다." |
| 구매 동기 | 고가 컨설팅까지는 부담스럽지만, 자녀에게 맞는 학교생활 설계 기준은 필요함 |
| 미리의 가치 | 고교 선택 기준, 미래 생기부 미리보기, 과목·활동·탐구 로드맵, 첫 30일 액션 |
| 검증 포인트 | 중3 학부모가 실제 위기감을 느끼는가, 첫 결과물이 고교 선택과 학교생활 설계에 도움이 된다고 느끼는가 |

## B. 고1 수시 포기 방지형 학부모

| 항목 | 내용 |
| --- | --- |
| 예시 이름 | 이서윤 학생 어머니 |
| 학부모 상황 | 고1 첫 중간고사 이후 내신과 생기부를 보고 수시를 계속 가져가야 할지 고민함 |
| 자녀 상황 | 수도권 일반고 고1, 중위권~중상위권, 진로가 아직 넓고 불명확함 |
| 핵심 감정 | "이미 늦은 건 아닌지 모르겠다. 지금이라도 무엇을 쌓아야 하는지 알고 싶다." |
| 구매 동기 | 수시를 포기하기 전에 과목·활동·탐구의 공백을 점검하고 싶음 |
| 미리의 가치 | 현재 활동 공백 진단, 세특 방향, 탐구 주제 추천, 학기 단위 회복 전략 |
| 검증 포인트 | 고1 5~7월 시점에 실제 결제 의향이 생기는가, 반복 리포트를 받을 이유가 있는가 |

## C. 비수도권 정보격차형 학부모

| 항목 | 내용 |
| --- | --- |
| 예시 이름 | 김하늘 학생 아버지 |
| 학부모 상황 | 지역 내 학종 정보와 합격 사례 접근성이 낮아 불안함 |
| 자녀 상황 | 비수도권 일반중·일반고, 성실하지만 입시 전략 정보가 부족함 |
| 핵심 감정 | "수도권 학생들은 다 아는 정보를 우리만 모르는 것 같다." |
| 구매 동기 | 지역에 상관없이 자녀 상황 기준의 판단 근거와 로드맵을 받고 싶음 |
| 미리의 가치 | 온라인 기반 개인화 진단, 모범 생기부 사례 기반 방향 제시, 학교생활 액션 추천 |
| 검증 포인트 | 정보 부족이 실제 지불 의사로 이어지는가, 온라인만으로 신뢰를 만들 수 있는가 |

## D. 특수진로 서사화형 학부모

| 항목 | 내용 |
| --- | --- |
| 예시 이름 | 최유진 학생 어머니 |
| 학부모 상황 | 자녀가 좋아하는 분야는 뚜렷하지만 이를 학교생활과 입시 서사로 연결하는 법을 모름 |
| 자녀 상황 | 중3~고2, 베이킹·발레·디자인·국제기구·창업 등 비전형 관심사를 가짐 |
| 핵심 감정 | "좋아하는 일이 있는 건 좋은데, 이게 대학과 생기부에 도움이 될 수 있을까?" |
| 구매 동기 | 자녀의 관심사를 포기시키지 않고 입시 경쟁력으로 바꾸고 싶음 |
| 미리의 가치 | 관심사를 과목·탐구·동아리·독서·수행평가 활동으로 번역 |
| 검증 포인트 | 특수진로가 충분한 시장 세그먼트가 되는가, 관련 데이터와 사례 확보가 가능한가 |

## E. 특목·프리미엄 비교형 학부모

| 항목 | 내용 |
| --- | --- |
| 예시 이름 | 정민준 학생 어머니 |
| 학부모 상황 | 외고·자사고·상위권 일반고 등 선택지가 많아 객관적 비교 기준을 원함 |
| 자녀 상황 | 수도권 중3, 상위권, 특목고 또는 특정 학과 진학을 고려함 |
| 핵심 감정 | "어느 학교를 가야 3년 뒤 입시 전략이 가장 좋아질까?" |
| 구매 동기 | 이미 많은 정보를 찾아봤지만 자녀 기준 시뮬레이션이 필요함 |
| 미리의 가치 | 고교 유형별 학교생활 시나리오, 과목·활동 로드맵 비교 |
| 검증 포인트 | 고가 컨설팅을 고려하는 상위권 학부모에게 미리가 충분히 차별적인가 |

## F. 월간 점검 니즈형 학부모

| 항목 | 내용 |
| --- | --- |
| 예시 이름 | 박세민 학생 어머니 |
| 학부모 상황 | 입시 정보를 찾아보지만 자녀가 실제로 잘 가고 있는지 확인할 방법이 없음 |
| 자녀 상황 | 중3~고3, 활동은 하고 있으나 진로와 생기부 방향이 명확히 연결되지 않음 |
| 핵심 감정 | "우리 아이가 지금 제대로 가고 있는지, 다음엔 뭘 해야 하는지 알고 싶다." |
| 구매 동기 | 성적표처럼 주기적으로 확인 가능한 학교생활 리포트를 원함 |
| 미리의 가치 | 월간·학기 요약 리포트, 활동 공백 알림, 다음 하루·주간·월간 액션 |
| 검증 포인트 | 학부모용 리포트가 구독 지속 이유가 되는가, 학생 사용성을 해치지 않고 공유할 수 있는가 |

## 1차 검증 추천

초기에는 아래 세 페르소나를 우선 비교한다.

| 우선 후보 | 이유 |
| --- | --- |
| A. 중3 진입 불안형 학부모 | 고교학점제와 고교 선택이 맞물려 미리의 핵심 서사가 가장 선명함 |
| B. 고1 수시 포기 방지형 학부모 | 실제 긴급도가 높고, 수시 포기 전 개입 가치가 큼 |
| F. 월간 점검 니즈형 학부모 | 결제자가 가장 뚜렷하며, 학부모용 리포트 가치 검증에 적합함 |

## 인터뷰에서 확인할 질문

| 검증 영역 | 질문 |
| --- | --- |
| 문제 강도 | 자녀의 고교 선택, 과목 선택, 생기부 방향 중 지금 가장 불안한 것은 무엇인가? |
| 기존 대안 | 현재 어떤 방식으로 정보를 찾고 있고, 무엇이 부족하다고 느끼는가? |
| 결과물 이해 | 미래 생기부 미리보기나 학교생활 로드맵을 보면 어떤 판단을 할 수 있을 것 같은가? |
| 결제 의향 | 무료 진단 이후 월간·학기 리포트와 액션 추천을 받는다면 비용을 낼 이유가 있는가? |
| 학생 사용성 | 자녀가 직접 입력해야 하는 부분이 어느 정도까지면 받아들일 수 있는가? |
