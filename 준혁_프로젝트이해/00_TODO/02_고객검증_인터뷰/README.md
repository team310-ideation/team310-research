## 먼저 검증할 가설 후보

| 가설 | 확인할 지표 |
| --- | --- |
| 중3~고3 학부모는 과목·활동·탐구 선택과 생기부로 이어지는 학교생활 설계에 실제 위기감을 느낀다. | 문제 강도, 현재 대안, 비용 지불 의향 |
| 미래 생기부 미리보기는 학생이 이해하고 학부모가 공유·결제 판단을 하게 만드는 강한 첫 결과물이다. | 결과물 이해도, 공유 의향, 신뢰도 |
| 하루/주간/월간/학기 액션은 첫 리포트 이후 재방문 이유가 된다. | 재사용 의향, 알림 수용도, 입력 지속 의향 |

## 인터뷰 원칙

- 가설을 먼저 쓰되, 가설을 직접 물어보지 않는다.
- 솔루션을 먼저 설명하지 않는다.
- 긍정 반응 뒤에는 가격 질문으로 진짜 반응을 확인한다.
- `좋아요`가 아니라 `재사용 의향`, `공유 의향`, `결제 의향`, `입력 지속 의향`을 구분해 기록한다.
