## 대표자 생년월일 반영 전략

| 항목 | 판단 |
| --- | --- |
| 생년월일 | 1988-09-12 |
| 현재 만 나이 | 2026-05-02 기준 만 37세, 2026-09-12에 만 38세 |
| 청년지원 창구 | `만 39세 이하` 청년창업자 조건은 2026년 공고 기준으로 안정권 |
| 시간 리스크 | 기준일을 실제 생일로 계산하는 공고는 2028-09-12부터 만 40세가 되어 청년 조건에서 빠질 수 있음 |
| 전략 결론 | SOMA 본과정 중 외부 정부지원사업 협약은 피하고, SOMA 종료 후 2027~2028년 청년창업사관학교·창업중심대학 청년 우선선발 창구를 우선 검토 |

- 2026년 청년창업사관학교 기본과정 공고는 `1986-01-31 이후 출생자`를 39세 이하 기준으로 보므로, 대표자 `1988-09-12`는 조건에 들어간다.
- 창업중심대학 지역기반 유형은 2026년에 선정기업의 60% 이상을 만 39세 이하 청년 창업자로 우선 선발한다고 발표되어, 대표자 생일 정보는 가점/우선선발 전략에 활용 가능하다.
- 2028년 이후 공고는 `공고일`, `신청 마감일`, `협약일` 중 어느 날짜를 나이 기준일로 삼는지 원문 확인이 필요하다.
- 2029년부터는 청년 트랙보다 `초기창업패키지`, `창업도약 전 단계`, `TIPS/민간투자 연계`, `교육·AI R&D` 쪽으로 포지셔닝을 바꾸는 것이 안전하다.
