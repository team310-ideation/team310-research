# MVP 3순위: 지식 확장형 꼬리 질문 로드맵

## 결론

`지식 확장형 꼬리 질문 로드맵`은 학생의 단발성 수행평가와 동아리 활동을 다음 탐구로 연결해 학종 준비의 연속성을 만든다. 핵심은 “다음에 무엇을 하면 좋을까”를 추천하되, 항상 **추천 근거**를 함께 보여주는 것이다.

## 사용자 문제

- 학생은 한 활동을 끝낸 뒤 다음 탐구로 확장하지 못한다.
- 활동이 많아도 서로 연결되지 않으면 학생부 서사가 약해진다.
- 학부모는 아이가 장기적으로 성장하고 있는지 보기 어렵다.

## 입력값

| 입력 | 예시 |
|---|---|
| 완료 활동 | `베이즈 정리 기반 진단 정확도 보고서` |
| 과목 | 확률과 통계 |
| 산출물 형태 | 보고서, 발표, 코드, 토론 |
| 학생 관심사 | 의료 AI, 데이터 분석 |
| 다음 마감일 | 2주 뒤 수행평가 |

## 출력값

| 출력 | 설명 |
|---|---|
| 다음 꼬리 질문 | 이전 활동에서 자연스럽게 이어지는 질문 |
| 심화 TODO | 조사, 비교, 실험, 발표 준비 등 다음 행동 |
| 연결 근거 | 이전 활동, 교과 개념, 학생 관심사의 연결 설명 |
| 난이도 | 쉬움/보통/어려움 |
| 예상 산출물 | 다음에 만들 수 있는 보고서, 발표, 그래프, 토론 자료 |

## 추천 근거 UI 예시

> 왜 이 TODO인가?
> - 이전 활동에서 `진단 정확도`를 다뤘습니다.
> - 현재 과목은 `확률과 통계`이고, 조건부확률 단원과 연결됩니다.
> - 희망 진로가 `의료 AI`이므로, 다음에는 `민감도/특이도와 오진 비용`을 비교하면 지식 확장 흐름이 자연스럽습니다.

## UX 원칙

- 한 번에 10개를 추천하지 않고, 다음 행동 1~3개만 보여준다.
- 학생이 직접 선택해야 다음 단계가 열린다.
- 추천 근거를 숨기지 않는다.
- 활동의 깊이를 우선하고, 활동 수 늘리기를 목표로 하지 않는다.

## 검증 지표

| 지표 | 목표 |
|---|---|
| 후속 TODO 선택률 | 40% 이상 |
| 후속 TODO 완료율 | 25% 이상 |
| 2회차 재방문율 | MVP 핵심 유지 지표 |
| 추천 근거 신뢰도 | “왜 추천됐는지 이해된다” 응답 70% 이상 |
