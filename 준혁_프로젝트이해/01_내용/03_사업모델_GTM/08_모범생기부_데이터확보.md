# 모범 생기부 데이터 확보 전략

## 왜 필요한가

- 모범 생기부 데이터셋은 범용 GPT가 갖기 어려운 데이터 우위가 될 수 있다.
- 미리의 추천 품질은 사용자 데이터와 모범 생기부 데이터가 함께 있어야 올라간다.
- 데이터 확보는 MVP 본기능과 분리해 별도 트랙으로 추진한다.

## 확보 후보

| 후보 | 장점 | 리스크/확인 필요 |
| --- | --- | --- |
| 합격생 생기부 구매 | 빠르게 사례 확보 가능 | 개인정보 동의, 활용 범위, 가격 검증 필요 |
| 소마 합격생 네트워크 | 접근성 있음 | 학종 합격자만 유효할 수 있음 |
| 자체 사용자 데이터 누적 | 제품 사용과 함께 축적 | 초기 품질이 낮고 시간이 걸림 |

## 회의 메모

- 합격생 생기부 구매 가격은 건당 3~7만 원, 연 40만 원 사례가 언급되었다.
- 단, 실제 구매 가능성과 법적 동의 범위는 별도 확인이 필요하다.
- 모범 생기부 RAG 데이터셋은 MVP에는 미포함하고 별도 추진하는 방향이 논의되었다.
