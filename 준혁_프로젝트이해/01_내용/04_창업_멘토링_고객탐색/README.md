## 핵심 요약

1. 고객 탐색은 세일즈보다 가설 검증이 먼저다.
2. 전체 서비스 범위는 중3~고3이지만, 6개월 안에 검증할 1차 진입 타겟은 뾰족하게 잡아야 한다.
3. 5월 MVP는 멘토링 매칭보다 AI 리포트와 웹 제품 중심으로 좁힌다.
4. 범용 AI와의 차별점은 학생 데이터 아카이빙, 입시 맥락 유지, 실행 체크리스트다.
5. 개인정보 동의서, 마스킹, 원본 저장 최소화는 베타 시작 전 필수다.
6. 휴포지토리의 첫 사업화 형태로 미리를 정의하고, 5월 30일 기획심의를 단기 목표로 둔다.
