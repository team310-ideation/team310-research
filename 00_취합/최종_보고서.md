# MIRI 최종 보고서: AI 학생부종합전형 코치 전략

## 1. 결론

MIRI는 `생기부 생성기`가 아니라 **학생부·활동·전형·일정을 근거 기반으로 진단하고, 이번 주 실행할 일을 쪼개주는 AI 학생부종합전형 프로젝트 매니저**로 포지셔닝해야 한다.

경쟁 환경을 보면 대형 입시 브랜드는 원서·합격예측·상담에 강하고, 과외/멘토링 플랫폼은 교과 매칭에 강하며, 해외 AI 입시 서비스는 에세이·원서·카운슬러 대시보드 중심이다. 한국 학종의 핵심인 `NEIS 학생부`, `세특`, `교사 관찰`, `전형별 요강`, `고교학점제 선택과목`을 학생 단위로 계속 관리하는 제품은 아직 빈 공간이 있다.

따라서 6개월 MVP는 `무료 학생부 진단 Lite → 3만 원대 전략 리포트 → 월간 AI 코치 → AI+검증 멘토 PM`으로 검증하고, V2에서는 `빈틈 전형`, `메디컬`, `예체능/체대`, `특목고`를 vertical로 확장하는 전략이 가장 현실적이다.

## 2. 시장과 경쟁 구도

| 경쟁군 | 대표 플레이어 | 강점 | 약점 | MIRI의 기회 |
|---|---|---|---|---|
| 대형 입시 브랜드 | 진학사, 메가스터디, 대형 컨설팅 센터 | 원서, 합격예측, 상담, 브랜드 신뢰 | 평소 학생부 활동을 매주 관리하는 UX는 약함 | 학생부 기반 진단과 행동 로드맵 |
| 학종/생기부 컨설팅 | 스터디코드, 메드스카이, 대치동 컨설팅, AI 생기부 분석 서비스 | 고관여 상담, 생기부 관리 수요 검증 | 가격이 높거나 1회성 리포트에 머물 수 있음 | 중간 가격대 AI+휴먼 구독 |
| 과외/멘토링 플랫폼 | 김과외, 설탭, 콴다과외, 숨고, 크몽 | 교과 과외 매칭, 튜터 풀, 빠른 연결 | 학종 실행 관리와 학생부 누적 관리가 약함 | 전공 멘토를 `학종 실행 PM`으로 재정의 |
| 해외 AI 입시 플랫폼 | CollegeVine, Cialfo, Scoir, MaiaLearning, BridgeU, SchooLinks, Crimson, AdmitYogi | 에세이, 대학 매칭, 카운슬러 대시보드, 원서 관리 | 한국 학종의 NEIS/세특 구조와 직접 맞지 않음 | 한국형 활동·전형 체크리스트 |
| AI 생산성 SaaS | Linear, Motion, Reclaim AI, Todoist, Notion, Asana, ClickUp, Trello | 할 일 쪼개기, 진행률, 우선순위, 협업 | 입시 데이터와 전형 맥락 없음 | `오늘 할 일 1개` UX를 입시에 이식 |
| 특수 진로/vertical | 메디컬 컨설팅, 예체능 입시 학원, 체대 입시 학원 | 전문성, 실기/면접/서류 노하우 | 정보가 흩어져 있고 전문가 의존도가 높음 | 서류·면접·환산표·일정 관리 보조 |

## 3. MIRI의 핵심 차별점

### 3.1 일반 ChatGPT 대비 차별점

| 항목 | 일반 ChatGPT | MIRI |
|---|---|---|
| 입력 데이터 | 사용자가 매번 맥락 설명 | 학생부, 활동 기록, 희망 전공, 전형 조건을 누적 |
| 결과물 | 범용 조언과 문장 생성 | 원문 근거가 붙은 진단, 로드맵, 면접 질문 |
| 실행 관리 | 대화 이후 추적 어려움 | 오늘/이번 주/이번 달 할 일과 진행률 추적 |
| 학부모 가치 | 결과를 신뢰하기 어려움 | 학부모 리포트, 위험 신호, 멘토 검수 큐 |
| 법무 안전성 | 대필·합격 보장 오해 가능 | 준비 상태, 누락 항목, 확인 필요 사항 중심 |

MIRI의 차별화 언어는 `AI가 생기부를 써준다`가 아니라 **AI가 학생부 근거를 읽고 다음 행동을 관리한다**여야 한다.

### 3.2 제품 언어

- 핵심 포지션: `AI 학생부종합전형 코치`
- 더 안전한 설명: `근거 기반 입시 준비 프로젝트 매니저`
- 학부모용 한 줄: `우리 아이 학생부에서 부족한 근거와 이번 달 해야 할 일을 보여주는 서비스`
- 학생용 한 줄: `내 희망학과에 맞춰 오늘 할 일과 면접 대비 질문을 알려주는 코치`
- 멘토용 한 줄: `AI가 만든 진단과 태스크를 검수하고 실행을 붙잡는 관리 도구`

## 4. 6개월 MVP 제안

### 결론

6개월 MVP는 거대한 입시 플랫폼이 아니라 **학생부 진단 Lite와 전공 로드맵을 중심으로 결제 가능성을 검증하는 얇은 제품**이어야 한다. 빈틈 전형, 멘토, 미래 생기부는 핵심 흐름에 붙는 옵션 또는 제한 파일럿으로 둔다.

### MVP 핵심 기능

| 우선순위 | 기능 | 내용 | 검증 지표 |
|---:|---|---|---|
| 1 | 학생부 진단 Lite | 학생부/활동 기록에서 강점, 약점, 부족 역량 표시 | 무료 진단 완료율, 결과 공유율 |
| 2 | 전공 적합도 | 희망학과와 현재 기록의 연결 근거 표시 | 유료 리포트 전환율 |
| 3 | 성장 로드맵 | 이번 학기 보완 활동, 탐구 주제, 수행평가 방향 제안 | 저장/실행 체크율 |
| 4 | 면접 예상 질문 | 학생부 근거 기반 질문과 꼬리질문 생성 | 질문 만족도, 멘토 피드백 품질 |
| 5 | 오늘 할 일 Focus View | 15~30분 단위의 오늘 할 일 1개 제시 | 주간 재방문율 |
| 6 | 학부모 요약 리포트 | 진행률, 위험 신호, 확인 질문 제공 | 학부모 공유/결제율 |

### 제한 파일럿 기능

| 기능 | 넣는 방식 | 주의점 |
|---|---|---|
| 미래 생기부 미리보기 | 로드맵을 따르면 기록이 어떻게 보일지 보여주는 참고 초안 | 공식 NEIS 기록처럼 보이면 안 됨 |
| 빈틈 전형 후보 | 지원 자격 체크와 준비 로드맵으로 제한 | “쉬운 전형” 또는 “합격 가능성 높음” 표현 금지 |
| 검증 멘토 피드백 | 5~10명의 수동 검증 멘토가 30분 화상 피드백 | 전국 매칭 플랫폼으로 시작하지 않음 |
| 메디컬/예체능 vertical | 일부 케이스 인터뷰와 컨시어지 실험 | 전문가 검수 전 자동 판정 금지 |

## 5. 가격과 수익화 가설

| 단계 | 가격 가설 | 제공 가치 | 목적 |
|---|---:|---|---|
| 무료 진단 Lite | 0원 | 학생부 섹션 감지, 강점/약점 맛보기, 누락 항목 | 리드 수집과 신뢰 형성 |
| 1회 전략 리포트 | 29,000~39,000원 | 전공 적합도, 보완 로드맵, 면접 질문 | 첫 결제 허들 검증 |
| 월간 AI 코치 | 월 49,000~99,000원 | 매월 기록 업데이트, 로드맵 갱신, 학부모 리포트 | 장기 사용 검증 |
| AI+전공 멘토 PM | 월 200,000~400,000원 | 격주/주간 멘토 체크, 탐구 실행 관리, 면접 피드백 | 대치동 고가 컨설팅과 AI 리포트 사이의 중간 시장 검증 |
| 전문가 검수 옵션 | 건당/월 별도 | 고위험 문구, 전형 전략, 법무·윤리 표현 검토 | 신뢰와 리스크 관리 |

❓ 가격은 실제 결제 데이터가 없으므로 가설이다. 멘토 미팅에서는 `3만 원대 리포트부터 검증할지`, `월간 코치로 바로 갈지`, `AI+전공 멘토 PM이 중산층 학부모에게 수용 가능한지`를 확인해야 한다.

## 6. 데이터·법무·운영 원칙

### 결론

MIRI는 학생 데이터를 회사가 오래 쌓아두는 서비스가 아니라, **학생/학부모가 업로드·삭제·내보내기·공유 권한을 통제하는 학생 데이터 Vault**로 설계해야 한다.

| 원칙 | 실행 방안 |
|---|---|
| 최소 수집 | MVP에서는 학생부/활동 기록 중심으로 받고, 사회통합·건강·장애·가족관계 증빙 원본은 받지 않음 |
| 분석 후 삭제 | 원본 학생부는 분석 후 삭제하고 결과 리포트만 선택 보관 |
| 권한 통제 | 학생/학부모가 멘토·컨설턴트에게 공유 범위를 선택 |
| 근거 표시 | 모든 추천은 원문 기록, 사용자 입력, 공개 모집요강 근거와 연결 |
| 자동화 고지 | AI 결과는 참고용이며 최종 판단은 학생·학부모·전문가가 한다고 표시 |
| 대필 오해 방지 | `NEIS 공식 생기부 생성`, `세특 대필`, `합격 보장` 표현 금지 |

### 금지하거나 뒤로 미룰 기능

- 합격 확률/합격 가능성 단정
- 공식 NEIS 생기부 생성처럼 보이는 기능
- 근거 없는 활동·수상·실적 생성
- 대학별 내부 평가 기준을 아는 것처럼 단정
- 미성년자 개인정보를 동의 없이 장기 저장
- 전국 멘토 매칭 플랫폼을 MVP 본체로 바로 구축

## 7. 확장 전략

### 7.1 빈틈 전형

빈틈 전형은 `쉬운 전형 추천`이 아니라 **지원 자격을 놓치기 쉬운 정원외·특수전형을 구조화하는 기능**으로 잡아야 한다.

| 범위 | 접근 |
|---|---|
| 초기 DB | 대학 30~50개, 전형군 20개, 자격조건 체크리스트 |
| 핵심 기능 | 전형 후보 발굴, 지원 자격 체크, 준비 서류, 마감일 |
| 피해야 할 표현 | “합격 가능성이 높다”, “아는 사람만 아는 쉬운 전형” |
| 검증 방식 | 모집요강 기반 체크리스트 정확도와 사용자 인터뷰 |

### 7.2 메디컬 vertical

메디컬은 지불 의향이 높을 가능성이 큰 프리미엄 후보지만, 합격 확률과 내부 평가 기준 단정은 피해야 한다.

| 기능 | 설명 |
|---|---|
| 생기부 무결성 진단 | 의학 계열 희망과 현재 활동의 연결성, 누락 근거 점검 |
| Deep Dive 트리 | 의학 주제 탐구가 얕게 끝나지 않도록 질문 구조화 |
| MMI 스파링 | 딜레마 질문, 꼬리질문, 논리 구조 점검 |
| 대학별 공개 평가 렌즈 | 공개 모집요강과 인재상 기반 체크리스트 |

### 7.3 예체능·체대 vertical

예체능은 AI가 실기를 판정하는 방향보다 **서류·면접·환산 계산·일정 관리 보조**가 안전하다.

| 분야 | MIRI가 들어갈 수 있는 기능 |
|---|---|
| 미대/예술 | 미술활동보고서, 작품 설명, 면접 질문 |
| 영화/영상/한예종 | 시놉시스 구조 점검, 제시문 면접 스파링 |
| 체대 | 실기·내신·수능 환산 체크, 목표 기록 역산 |
| 음대/공연 | 레슨/콩쿠르/실기 일정 정리, 면접 대비 |

### 7.4 특목고·고입 버전

특목고 정보는 수요가 있지만 현재 MIRI의 메인 타겟인 `고등학생 학종 코치`와 바로 붙이면 타겟이 흔들릴 수 있다. V2에서 `고입 버전 MIRI` 또는 `중3 진로·고입 로드맵`으로 분리하는 편이 안전하다.

## 8. 발표용 핵심 문장

- “미리의 빈 공간은 생기부 AI 리포트가 아니라, AI가 진단하고 사람 멘토가 실행을 붙잡아주는 학종 프로젝트 관리입니다.”
- “대형 입시 브랜드는 원서와 합격예측에 강하고, 과외 플랫폼은 교과 수업에 강합니다. 미리는 그 사이에서 학생부 근거와 이번 주 행동을 연결합니다.”
- “우리는 합격 확률을 팔지 않고, 준비 상태·누락 항목·다음 행동을 팝니다.”
- “MVP는 전국 멘토 매칭 플랫폼이 아니라 학생부 진단 리포트와 수동 멘토 피드백을 묶은 컨시어지 실험으로 시작합니다.”
- “미래 생기부 미리보기는 공식 기록 생성이 아니라 로드맵 결과를 이해시키는 참고 화면으로 제한합니다.”

## 9. 멘토에게 물어볼 것

1. ❓ 첫 유료 상품은 `3만 원대 1회성 전략 리포트`가 맞는가, 아니면 `월 5만~10만 원 AI 코치`가 맞는가?
2. ❓ MIRI의 첫 포지션은 `AI 학생부 리포트`보다 `입시 준비 프로젝트 매니저`가 더 설득력 있는가?
3. ❓ 합격 예측 없이 `준비 상태/누락 항목/오늘 할 일`만으로 학부모 결제가 가능한가?
4. ❓ 빈틈 전형 매칭은 MVP에 넣을지, 학생부 코치 V2 차별화 기능으로 둘지?
5. ❓ 초기 멘토 기능은 전국 매칭이 아니라 5~10명 검증 멘토의 30분 피드백 실험으로 충분한가?
6. ❓ AI기본법상 교육 분야 고영향 AI 오해를 피하려면 어떤 표현과 고지 문구가 필요한가?

## 10. 최종 실행안

| 기간 | 할 일 | 산출물 |
|---|---|---|
| 1~2주 | 샘플 학생부/활동 기록으로 진단 로직 검증 | 진단 카드, 근거 표시 예시 |
| 3~4주 | 전공 적합도와 로드맵 리포트 제작 | 3만 원대 전략 리포트 샘플 |
| 5~6주 | 학부모 10명, 학생 10명 인터뷰 | 결제 의향, 불신 지점, 문구 피드백 |
| 7~8주 | 수동 멘토 피드백 실험 | AI 리포트 + 30분 피드백 패키지 |
| 9~12주 | 무료 진단 Lite 랜딩과 카카오 채널 퍼널 | 리드 수집, 전환율, 재방문율 |
| 13~24주 | 월간 AI 코치와 학부모 리포트 검증 | 월 구독 유지율, 실행률, 재구매율 |
