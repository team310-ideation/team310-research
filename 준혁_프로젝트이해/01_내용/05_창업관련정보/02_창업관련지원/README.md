## 결론

지금 대표자 조건만 보면 `예비창업패키지`가 가장 정합성이 높지만, AI·SW마에스트로 본과정과 기간이 겹치면 중복 수혜 불가다.

대표자 생년월일은 `1988-09-12`로 확인되었다. 2026-05-02 기준 만 37세이고, 2026년 청년창업사관학교·창업중심대학의 `만 39세 이하` 청년 조건에는 안정적으로 들어간다.

`청년창업사관학교`와 `창업중심대학 예비창업자 유형`도 정부지원 창업 프로그램이므로, SOMA 본과정 중 동시 참여는 위험하다. 현실적인 순서는 `SOMA 본과정 종료 → 2027~2028년 예비창업패키지/청년창업사관학교/창업중심대학 신청`이다.

다만 실제 생일 기준으로 계산하는 공고는 2028-09-12부터 만 40세가 되어 청년 조건에서 빠질 수 있다. 2028년 공고는 `공고일`, `신청 마감일`, `협약일` 중 무엇을 나이 기준일로 삼는지 원문 확인이 필요하다.

반대로 `텍스코어(TeX-Corps)`는 멘토링 메모에는 강하게 추천되어 있지만, 현재 대표자 정보만으로는 공공기술·실험실창업 기반 요건이 약해서 바로 넣기 어렵다.

VC 투자유치 단계별 기준은 [투자단계별 평가기준](../03_투자유치_단계/01_투자단계별_평가기준.md)에 별도로 정리한다.
