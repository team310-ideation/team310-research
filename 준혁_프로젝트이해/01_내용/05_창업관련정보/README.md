## 정리 기준

- 대표자 조건: 1988-09-12 출생, 대학졸업자, 사업자등록 전, 과기부 소속 소프트웨어마에스트로 참가자
- 우선 판단: 예비창업자 지원 가능 여부, 만 39세 이하 청년 조건, 사업자등록 필요 여부
- 별도 확인: 과거 사업자등록 이력, SOMA 본과정과 외부 창업지원사업 기간 중복 여부, 2028년 이후 공고별 나이 기준일
- 사용자 확인 반영: 동일 아이템으로 받은 정부지원금·사업화자금은 현재 없는 것으로 정리
- 투자유치 단계는 프리시드, 시드, 프리A, 시리즈A로 구분해 관리한다.
