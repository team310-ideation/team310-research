## 본인 미션 (이 3가지만 답하면 됨)

> 1차 멘토님이 *"여기가 진짜 차별점이 될 것"* 이라고 짚은 영역. 이 트랙의 결과물이 2차 미팅에서 가장 강력한 무기가 될 가능성 큼.

### 1. 빈틈 전형 데이터 조사 (90분)

찾아야 할 것:
- 한국 대학 입시의 **잘 알려지지 않은 정원외/특수 전형** 사례 20개 이상
  - 외국인 정원외 전형
  - 재외국민 전형
  - 농어촌학생 전형
  - 특수재능자 전형
  - 학교생활우수자 전형 (대학별 조건 다름)
  - 한두 명만 뽑는 학과·전형
- 이 전형들의 **데이터를 어디서 모을 수 있는가**?
  - 대학교 입학처 홈페이지 크롤링 가능?
  - 한국대학교육협의회 자료 활용?
  - 사설 입시 사이트의 정보?

→ 산출물: `01_빈틈전형_데이터_조사.md`

### 2. 특수 진로 트랙 — 한예종/체육 특기 (60분)

1차 멘토가 *"하프 같은 거 가지고 있으면 들어가는 학교 있다"* 라고 짚은 영역. 시장 검증:
- 한예종, 한국예술원, 서울예대 등 예술 특기 진학 경로
- 체육 특기자 전형 (대학별)
- 어떤 악기·종목이 정원이 적어 진입 쉬운가
- 이 분야 컨설팅의 현재 시장 가격
- 1타 강사·전문가 매칭 가능성

→ 산출물: `02_특수진로_한예종_체육.md`

### 3. 특목고 정보 (60분)

1차 멘토가 *"특목고에 대한 데이터를 제공하는 업체가 거의 없다"* 라고 짚은 정보 부족 영역:
- 외고/과학고/영재고/자사고 진학 정보
- 학부모가 가장 답답해하는 부분 (입시 일정, 합격 기준, 면접 등)
- 우리가 어떤 정보를 제공할 수 있을지
- 이 영역에 진입한 회사 / 안 한 이유

→ 산출물: `03_특목고_정보_부족영역.md`

### 4. 결론 1페이지 (30분, 마지막에)

위 3개를 한 페이지로 압축. `templates/결론_1페이지_템플릿.md` 사용.

→ 산출물: `00_결론_1페이지.md`
