## 정리 기준

- 시장규모: 전체 시장이 아니라 우리가 실제 접근하고 수익화할 수 있는 시장까지 좁힌다.
- 경쟁사: 회사 이름보다 고객군, 가격대, 이용 목적을 기준으로 다시 묶는다.
- 고객 인터뷰: 세일즈보다 가설 검증을 먼저 한다.
- 타겟 세그먼트: 학교 전략, 지역, 성적대, 시즌성을 기준으로 나눈다.
