# 07_홍순엽_VC멘토링_회의안건

작성 기준: 2026-05-02(KST)  
회의 예정일: 2026-05-05  
멘토: 홍순엽 VC투자자 멘토님

## 회의 목표

- [ ] 미리(MIRI)가 VC 관점에서 투자 가능한 사업으로 보이는지 1차 판단 받기
- [ ] 5월 30일 기획심의 자료에서 반드시 강조해야 할 투자자 관점 메시지 정리
- [ ] 시장성, 차별화, 수익모델, 초기 검증 지표 중 가장 약한 부분 확인
- [ ] 창업 트랙 진입 전까지 만들어야 할 핵심 마일스톤 3개 확정

## 60분 회의 진행안

### 0~5분: 현재 상황 공유

- [ ] 팀 310 소개와 현재 단계 설명
- [ ] 미리(MIRI) 한 줄 정의 공유
- [ ] 5월 30일 기획심의가 단기 목표임을 설명

핵심 질문:

- [ ] 현재 단계에서 VC 멘토님께 가장 먼저 검증받아야 할 사업 쟁점은 무엇인가?

### 5~12분: 1분 피치 + 서비스 핵심 설명

- [ ] 문제: 고교학점제 이후 중3·고1 학생/학부모가 3년 뒤 생기부와 선택 경로를 그리지 못함
- [ ] 해결책: 초·중 생기부와 진로 인터뷰 기반 `미래 생기부 미리보기` + 3년 액션 설계
- [ ] 고객: 결제자는 학부모, 사용자는 학생
- [ ] 초기 제품: AI 리포트/웹 제품 중심, 멘토 매칭은 후순위 또는 보조 기능

핵심 질문:

- [ ] 이 피치가 투자자 관점에서 바로 이해되는가?
- [ ] "생기부 관리 서비스"가 아니라 "학생 의사결정 OS"라는 포지셔닝이 설득력 있는가?
- [ ] `미래 생기부 미리보기`라는 표현이 강점인지, 과장/리스크로 보이는지?

### 12~22분: 시장성과 타겟 검증

- [ ] 1차 타겟 후보: 중3 → 고1 진학 예정자, 중3 학부모, 고1 학부모
- [ ] 초기 유망 세그먼트: 중위권 수도권 일반고, 비수도권 정보 부족 학생, 학종 중심 학교 학생
- [ ] 시장 변화 근거: 고교학점제, 2030학년도 대입 변화, 생기부/세특/탐구 서사 중요도 증가

핵심 질문:

- [ ] VC 관점에서 이 시장은 충분히 크고 성장하는 시장으로 보이는가?
- [ ] TAM/SAM/SOM을 어떤 기준으로 잘라야 투자자에게 설득력 있는가?
- [ ] "중3 학부모는 실제 위기감이 있는가"를 어떤 데이터로 보여줘야 하는가?
- [ ] 중3부터 시작하는 시장 진입이 맞는가, 아니면 고1/고2가 더 결제 가능성이 높은가?

### 22~32분: 경쟁사와 차별화

- [ ] 비교군: 대치동 고가 컨설팅, 고3 원서 플랫폼, 3년 풀 관리 컨설팅, 대학생 과외 O2O, 범용 AI
- [ ] 미리의 후보 차별점: 학생 데이터 아카이빙, 모범 생기부 데이터셋, AI 리포트 갱신, 멘토 운영구조
- [ ] 범용 ChatGPT 대비 차이: 학생의 누적 데이터와 입시 맥락을 계속 보존·해석한다는 점

핵심 질문:

- [ ] 투자자 입장에서 가장 강한 차별화 포인트는 무엇으로 잡아야 하는가?
- [ ] `학생 데이터 아카이빙`은 충분한 해자(moat)로 보이는가?
- [ ] 모범 생기부 RAG 데이터셋은 지금부터 확보해야 할 핵심 자산인가, 아니면 초기에는 과한가?
- [ ] 기존 대치동 컨설팅/진학사/바이브온/스터디코드와 비교할 때 가장 취약한 지점은 무엇인가?

### 32~42분: 수익모델과 GTM

- [ ] 초기 전략 후보: 무료 베타로 데이터 확보 후 유료 전환
- [ ] 가격 가설: 월 3만 원 또는 학기 단위 패키지
- [ ] 유료 전환 가치: 단일 리포트가 아니라 지속 갱신, 활동 아카이빙, 학부모용 진행 리포트
- [ ] 유입 채널: 맘카페, 네이버 카페, 당근마켓, 학교 방문, 세종 클래스 유튜브, 기존 유저 추천

핵심 질문:

- [ ] 초기에는 무료 베타가 맞는가, 아니면 소액 결제로 지불 의사를 바로 검증해야 하는가?
- [ ] 월 3만 원 모델이 투자자 관점에서 너무 작은 ARPU로 보이는가?
- [ ] 학기 패키지, 월 구독, 리포트 1회 결제 중 어떤 가격 구조가 더 검증하기 좋은가?
- [ ] 베타 유저 10명, 100명 단계에서 각각 봐야 할 핵심 지표는 무엇인가?
- [ ] B2C 학부모 채널로 시작하는 것이 맞는가, 아니면 학교/학원/B2B2C 채널을 먼저 검토해야 하는가?

### 42~52분: MVP 범위와 검증 지표

- [ ] 5월 MVP 후보: 미래 생기부 미리보기, AI 리포트, 30일 액션, 활동 아카이빙 초안
- [ ] 후순위 후보: 멘토 매칭, 커뮤니티, 프롬프트 교육, 학부모 대시보드, 학교 B2B
- [ ] 베타 검증 핵심: 문제 공감, 결과물 공유 의향, 결제 의향, 재방문/재사용 의향

핵심 질문:

- [ ] 5월 안에 투자자/심사위원에게 보여줄 MVP는 어디까지면 충분한가?
- [ ] 멘토 매칭을 빼면 실행력을 만드는 서비스로 보이지 않을 위험이 있는가?
- [ ] 반대로 멘토 매칭을 넣으면 확장성 없는 컨설팅 사업으로 보일 위험이 큰가?
- [ ] PMF 전 단계에서 반드시 수집해야 할 정량/정성 지표는 무엇인가?

### 52~60분: 투자자 관점 피드백과 다음 액션

- [ ] 5월 30일 기획심의 자료에서 가장 먼저 고쳐야 할 장표 확인
- [ ] 투자자가 보기 싫어할 표현, 과장되어 보이는 표현 확인
- [ ] 창업 트랙 진입 전까지 필요한 마일스톤 정리

핵심 질문:

- [ ] 이 사업이 지금 VC가 볼 만한 형태가 되려면 어떤 증거가 더 필요한가?
- [ ] 6월 말까지 만들면 좋은 traction 지표 3개는 무엇인가?
- [ ] 지금 단계에서 투자 유치보다 정부지원/엑셀러레이팅/파일럿 중 어디에 집중해야 하는가?

## 시간이 남으면 물어볼 후속 질문

- [ ] 교육/입시/청소년 AI 서비스에서 투자자가 가장 민감하게 보는 법적·평판 리스크는 무엇인가?
- [ ] `미래 생기부 미리보기`가 대필 도구처럼 보이지 않게 하려면 어떤 UX/카피 방어선이 필요한가?
- [ ] 해외 한국 입시 준비생, 교포, 주재원 자녀를 장기 비전으로 적어도 되는가?
- [ ] 학교 전산 시스템/B2G 확장은 장기 비전으로만 둬야 하는가, 초기부터 파일럿을 찾아야 하는가?
- [ ] 대학생 멘토 네트워크는 해자인가, 운영 부담인가?

## 회의 전 준비물

- [ ] 미리 서비스 한 줄 정의
