# 06_멘토_질문리스트

작성 기준: 2026-05-02(KST)

## P0: 다음 멘토링에서 꼭 물어볼 것

- [ ] 5월 MVP 범위는 `AI 리포트/웹 제품` 중심으로 좁히는 것이 맞는가, 아니면 최소한의 `대학생 멘토 매칭`도 함께 보여줘야 하는가?
- [ ] 6개월 안에 가장 뾰족하게 잡아야 할 1차 타겟은 `중3 학생`, `중3 학부모`, `고1 학생`, `고1 학부모` 중 어디인가?
- [ ] `미래 생기부 미리보기`를 제품 핵심 가치로 밀 때, 멘토님 기준에서 가장 위험한 오해나 설득 약점은 무엇인가?
- [ ] AI 리포트와 대학생 멘토 매칭의 역할을 어떻게 분리해서 설명해야 "제품도 애매하고 컨설팅도 애매한" 인상을 피할 수 있는가?
- [ ] 첫 베타 유저 10명을 모을 때, 어떤 세그먼트를 반드시 섞어야 PMF 검증에 의미가 있는가? 예: 수도권/비수도권, 성적대, 학종 학교/정시 학교, 진로 확정/미정.
- [ ] 다음 고객 인터뷰에서는 어떤 가설 3개를 먼저 검증해야 하는가? 특히 "중3 학부모가 실제 위기감을 느끼는가"를 어떻게 물어봐야 하는가?
- [ ] 초기에는 무료 베타가 맞는가, 아니면 월 3만 원 같은 낮은 가격으로 결제 의향을 바로 검증해야 하는가?
- [ ] 모범 생기부 데이터셋과 합격생 생기부 구매는 지금 별도 트랙으로 시작해도 되는가, 아니면 법무 검토 전까지 멈춰야 하는가?
- [ ] 생기부·활동 데이터 저장 시 베타 단계에서 반드시 지켜야 할 개인정보 레드라인은 무엇인가? 저장 금지, 마스킹, 동의서, 보유기간 기준을 확인하고 싶다.
- [ ] SOMA 본과정과 외부 창업지원사업 협약/지원기간이 겹칠 때, 5월 30일 기획심의 자료에는 어떤 지원사업 전략으로 적는 것이 안전한가?

## P1: 기획서·발표 전 검증할 것

- [ ] `찐로/미리/휴포지토리` 브랜드 구조를 어떻게 설명해야 서비스가 산만해 보이지 않는가?
- [ ] 경쟁사를 `대치동 고가 컨설팅`, `고3 원서 플랫폼`, `3년 풀 관리 컨설팅`, `대학생 과외 O2O`, `범용 AI`로 나누는 방식이 설득력 있는가?
- [ ] 다수 학생 대상 합리적 가격 모델에서 미리의 진짜 차별점은 `AI`, `아카이빙`, `모범 생기부 데이터`, `멘토 관리` 중 무엇으로 잡아야 하는가?
- [ ] 고교학점제와 2030학년도 대입 변화를 시장 진입 타이밍으로 제시할 때, 기획서에 어떤 근거 수치를 우선 넣어야 하는가?
- [ ] 학부모용 월간/단계별 리포트가 결제 유지에 실제로 도움이 될까? 어떤 항목을 담아야 "성적표 같은 체감 가치"가 생기는가?
- [ ] 대학생 멘토의 전문성 부족 우려를 멘토 교육 커리큘럼과 객관적 입시 좌표로 해결한다는 설명이 충분한가?
- [ ] 담당 멘토와 핀포인트 멘토를 나눌 경우, MVP에서 먼저 구현하거나 설명해야 할 최소 운영 구조는 무엇인가?
- [ ] 맘카페, 네이버 카페, 당근마켓, 학교 방문, 세종 클래스 유튜브 중 베타 유저 10명 확보에 가장 먼저 쓸 채널은 무엇인가?
- [ ] 교사 인터뷰는 고객 인터뷰와 별도 트랙으로 가져갈 가치가 있는가? 있다면 담임/진로교사에게 무엇을 물어봐야 하는가?
- [ ] 5월 30일 창업 트랙 심의 자료에서 시장 규모(TAM/SAM/SOM)는 어느 정도 깊이까지 계산해야 충분한가?

## P2: 후속 멘토링에서 다룰 것

- [ ] 멘토 매칭을 부가 수익으로 붙일 때 수수료, 멘토 시급, 플랫폼 이용료를 어떻게 나눠야 공급자와 수요자 모두 납득하는가?
- [ ] 무료 공개형 세종 클래스와 유료 미리 서비스를 어떻게 연결해야 무료 콘텐츠가 유료 전환을 방해하지 않는가?
- [ ] AI 활용 능력 교육을 핵심 차별점으로 가져갈 경우, 입시 서비스가 아니라 AI 교육 서비스처럼 보이는 위험은 없는가?
- [ ] 대치동 3년 풀 관리 컨설팅의 가격·제공 범위를 어디까지 조사해야 IR 자료에서 비교 근거로 쓸 수 있는가?
- [ ] 해외 한국 입시 준비생, 교포, 주재원 자녀를 장기 타겟으로 적어도 되는가, 아니면 지금 단계에서는 집중력을 해치는가?
- [ ] 학교 전산 시스템 또는 B2G 진출 가능성은 장기 비전으로만 두는 것이 맞는가, 아니면 초기부터 파일럿 파트너를 찾아야 하는가?
- [ ] `미래 생기부 미리보기`가 줄글 생성으로 보일 경우 대필 도구 리스크가 있는데, 결과물을 개요·키워드·질문형으로 바꾸는 방향이 맞는가?
- [ ] 일반 트랙 종료일 메모의 `11월 31일`은 실제 어떤 날짜를 의미하는가?
- [ ] 맥미니/맥스튜디오 메모는 개발 장비 검토인지, 서비스 운영 인프라 검토인지 확인이 필요하다.
