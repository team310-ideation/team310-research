# 부모 광고 채널

## 한 줄 결론

한국 학부모 타겟은 **맘카페/입시카페 신뢰 → 카카오톡 채널 리텐션 → 네이버·인스타·유튜브 광고 확장 → 학원 제휴 전환** 순서가 현실적이다. 처음부터 대형 광고를 태우기보다, 무료 진단 Lite를 미끼로 `학부모 리드`를 모으고 카카오톡 채널/문자/상담으로 전환해야 한다.

## 채널별 판단표

| 채널 | 역할 | 장점 | 단점 | 초기 우선순위 |
|---|---|---|---|---|
| 맘카페/입시카페 | 신뢰·입소문 | 학부모 고민이 실제로 모임 | 광고 티가 나면 역효과 | 1순위 |
| 네이버 검색/블로그 | 니즈 포착 | “학생부 진단”, “학종 컨설팅” 검색 수요 | 키워드 경쟁 가능 | 1순위 |
| 카카오톡 채널 | 리텐션/상담 | 학부모가 익숙함, 메시지 재접촉 가능 | 친구 확보 전에는 도달 약함 | 1순위 |
| 인스타그램 | 문제 인식/브랜딩 | 카드뉴스·숏폼 확산 | 결제 의도 낮을 수 있음 | 2순위 |
| 유튜브/쇼츠 | 신뢰 구축 | 입시 설명 콘텐츠에 적합 | 제작 비용/시간 필요 | 2순위 |
| 학원 제휴 | B2B 전환 | 신뢰와 학생 풀 확보 | 영업 사이클 길다 | 2순위 |
| 오프라인 설명회 | 고가 전환 | 신뢰 강함 | 운영 부담 큼 | 후순위 |

## 채널 1. 맘카페/입시카페

### 왜 중요함

입시·학생부는 검색광고보다 “다른 엄마가 써봤다”는 신뢰가 강하다. 특히 학생부종합전형은 정보 비대칭이 크기 때문에 학부모 커뮤니티에서 불안이 모인다.

### 실행 방식

- 광고글보다 “무료 학생부 진단 체크리스트” 배포
- 댓글/쪽지로 리드 확보
- 지역 맘카페는 강남/목동/분당/판교/대구수성/부산해운대 등 교육열 높은 지역부터
- 입시카페는 고1/고2 학부모 게시판 중심

### 콘텐츠 예시

- “고1 학생부, 지금부터 망한 건지 아닌지 보는 체크리스트”
- “학생부종합전형에서 활동보다 중요한 건 기록의 연결성”
- “희망학과는 있는데 생기부가 안 따라올 때 보는 5가지”

### 리스크

- 바이럴 조작처럼 보이면 역효과
- 카페 규정 위반 시 삭제/강퇴
- 학부모 불안 자극형 문구는 브랜드 신뢰 하락

## 채널 2. 네이버 검색/블로그/성과형 디스플레이

네이버 성과형 디스플레이 광고는 실시간 입찰과 성별·연령·지역·관심사 등 타겟팅을 제공한다. 학부모 타겟은 네이버 검색/블로그/카페 소비가 강하기 때문에 초기 실험 채널로 적합하다.

### 검색 키워드 가설

| 키워드군 | 랜딩 메시지 |
|---|---|
| 학생부 진단 | “내 아이 학생부 강점/약점 무료 확인” |
| 학종 컨설팅 | “컨설팅 전에 AI로 먼저 진단” |
| 세특 관리 | “이번 학기 세특 소재 로드맵” |
| 희망학과 생기부 | “전공 적합도와 부족 요소 확인” |
| 면접 예상 질문 | “학생부 기반 면접 질문 생성” |

### 운영 방식

- 검색광고: 고의도 키워드
- 블로그 SEO: 학부모 교육 콘텐츠
- 성과형 디스플레이: 무료진단 리타겟팅

## 채널 3. 카카오톡 채널

카카오톡 채널은 신규 유입보다 리텐션과 상담 전환에 좋다. 카카오 비즈니스 가이드는 채널 메시지 집행 시 운영정책과 광고성 정보 가이드를 준수해야 한다고 안내한다.

### 미리 활용 방식

1. 랜딩페이지에서 “무료 진단 결과 카톡으로 받기”
2. 카카오톡 채널 추가
3. 진단 결과 요약 발송
4. 3일 뒤 로드맵 예시 발송
5. 7일 뒤 유료 전략 리포트 제안

### 메시지 예시

```text
[미리] 학생부 진단 Lite 결과가 준비됐어요.
강점 3개, 부족 요소 2개, 이번 학기 보완 방향을 확인해보세요.
```

### 리스크

- 광고성 정보 수신동의 필요
- 과도한 메시지는 차단 유도
- 교육/입시 불안 자극 문구 주의

## 채널 4. 인스타그램

인스타그램은 즉시 결제보다 문제 인식과 공유에 좋다. 학생보다 학부모 타겟 콘텐츠는 카드뉴스, 릴스, 짧은 진단 예시가 맞다.

### 콘텐츠 예시

- “고1 학생부에서 놓치면 큰 3가지”
- “컴공 희망인데 생기부에 수학만 있고 프로젝트가 없을 때”
- “면접에서 반드시 물어볼 학생부 구멍”

### KPI

- 저장률
- 프로필 클릭률
- 무료 진단 신청률
- 카카오톡 채널 추가율

## 채널 5. 유튜브/쇼츠

유튜브는 긴 설명과 신뢰 형성에 적합하다. Google Ads 도움말에 따르면 동영상 캠페인은 판매, 리드, 웹사이트 트래픽 등 목표별로 운영할 수 있고 YouTube 및 Google 동영상 파트너에 광고를 게재할 수 있다.

### 콘텐츠 포맷

- 60초 쇼츠: “학생부종합전형에서 진짜 보는 것”
- 5분 설명: “전공적합도는 어떻게 생기부에 남는가”
- 라이브: “고1/고2 학생부 무료 진단 사례”

## 채널 6. 학원 제휴

학원 제휴는 초반 광고 효율보다 B2B 신뢰에 가깝다.

### 제휴 제안

- 학원생 10명 무료 진단
- 학원용 리포트 로고 삽입
- 컨설턴트 검수 옵션
- 학생부 진단 설명회 공동 개최

## 추천 퍼널

```text
맘카페/네이버/인스타/유튜브
→ 무료 학생부 진단 Lite
→ 카카오톡 채널 추가
→ 진단 결과 일부 공개
→ 29,000~39,000원 전략 리포트
→ 99,000~199,000원 코치 패키지
→ 학원/B2B 제휴
```

## 첫 2주 광고 실험안

| 실험 | 예산 | 목표 | 성공 기준 |
|---|---:|---|---|
| 맘카페 콘텐츠 10개 | 0~30만 원 | 댓글/쪽지 리드 | 리드 20명 |
| 네이버 검색 소액 | 30만 원 | 무료진단 신청 | 신청 CPA 15,000원 이하 |
| 인스타 카드뉴스 | 20만 원 | 저장/프로필 클릭 | 저장률 3% 이상 |
| 카카오 채널 | 0~10만 원 | 리텐션 | 친구 추가 후 결과 확인 50% |

## 멘토에게 물어볼 것

- ❓ 초기 고객은 고1/고2 학부모가 맞는가, 고3 면접 대비 학부모가 더 빠른가?
- ❓ 유료 전환은 카카오톡 상담형이 맞는가, 결제 페이지 직접 전환이 맞는가?
- ❓ 맘카페 바이럴을 내부에서 직접 할지, 학부모 체험단을 모집할지?
