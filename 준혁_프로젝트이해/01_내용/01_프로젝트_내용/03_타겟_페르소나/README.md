# 03_타겟_페르소나

## 바로가기

- [페르소나](01_페르소나.md)

## 역할

- 초기 타겟을 여러 학부모 후보 페르소나로 나누어 비교한다.
- 페르소나의 주어는 학부모로 두고, 학생은 자녀 상황과 실제 사용 맥락으로 함께 관리한다.
- 학년, 학교 유형, 지역, 진로 불확실성, 학교생활 설계 니즈를 학부모의 불안과 구매 판단 기준으로 정리한다.
- 시장조사 문서의 세그먼트와 연결해 실제 인터뷰 대상을 정한다.
- 현재 우선 비교 후보는 `중3 진입 불안형 학부모`, `고1 수시 포기 방지형 학부모`, `월간 점검 니즈형 학부모`이다.
