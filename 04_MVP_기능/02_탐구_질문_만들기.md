# MVP 2순위: 탐구 질문 만들기

## 결론

`탐구 질문 만들기`는 학생이 수업 중 스스로 사고하고 질문하는 모습을 만들기 위한 기능이다. 정답이나 완성문을 주는 것이 아니라, 교과 개념을 더 깊이 이해하기 위한 **질문 후보와 사고 방향**을 제공한다.

## 왜 필요한가

2026학년도 체제에서는 수행평가가 수업 중 실시되는 방향으로 강화되고, 교사의 직접 관찰이 더 중요해진다. 따라서 학생이 수업 안에서 던질 수 있는 질문, 토론거리, 후속 조사 방향을 준비하는 기능이 안전하고 실용적이다.

## 입력값

| 입력 | 예시 |
|---|---|
| 과목/단원 | 영어 독해, 생활과 윤리, 확률과 통계 |
| 수업 내용 | `의료 AI 기사 읽기`, `개인정보와 윤리`, `베이즈 정리` |
| 학생 관심사 | 의료 AI, 데이터 분석, 공학, 심리 |
| 질문 난이도 | 기본 / 심화 / 발표용 |

## 출력값

| 출력 | 설명 |
|---|---|
| 탐구 질문 3~5개 | 학생이 직접 고를 질문 후보 |
| 질문 의도 | 이 질문이 어떤 교과 역량을 보여주는지 |
| 조사 키워드 | 학생이 직접 찾아볼 검색어 |
| 수업 활용 방식 | 발표, 토론, 질의응답, 보고서 중 어디에 쓸지 |
| 주의점 | 억지 진로 연결이나 과목 본질 이탈 경고 |

## 좋은 질문 기준

- 교과 단원의 핵심 개념과 연결된다.
- 학생이 직접 자료를 찾고 생각을 전개할 수 있다.
- 진로와 연결되더라도 과목 본연의 역량을 해치지 않는다.
- 단순 찬반보다 이유, 조건, 한계, 비교를 묻게 한다.

## 예시

| 입력 | 질문 예시 |
|---|---|
| 확률과 통계 + 의료 AI | `진단 모델의 정확도보다 민감도와 특이도가 더 중요한 상황은 언제인가?` |
| 생활과 윤리 + 개인정보 | `공익을 위한 의료 데이터 활용은 개인 동의 없이 어디까지 허용될 수 있는가?` |
| 영어 + AI 기사 | `영문 기사에서 AI의 가능성과 위험을 표현하는 단어 선택은 어떤 태도를 만든다?` |

## 절대 하지 말 것

- 답변 본문을 대신 작성하지 않는다.
- “이 질문을 쓰면 세특이 좋아진다”처럼 말하지 않는다.
- 모든 과목을 억지로 진로에 붙이지 않는다.
- 교과 수준을 벗어난 대학 전공 지식을 무리하게 제시하지 않는다.

## 검증 지표

| 지표 | 목표 |
|---|---|
| 질문 저장률 | 50% 이상 |
| 질문 수정률 | 학생이 자기 말로 고친 비율 측정 |
| 수업 활용 보고 | 실제 발표/질문/보고서에 활용했다는 응답 |
| 교사 거부감 | 인터뷰에서 대필 도구로 보이는지 확인 |
