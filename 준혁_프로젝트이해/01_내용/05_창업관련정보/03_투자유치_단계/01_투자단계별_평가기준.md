# 투자단계별 평가기준

작성 기준: 2026-05-02(KST)

## 결론

미리(MIRI)는 현재 **지원금/부트스트랩 단계에서 프리시드로 넘어가기 전**에 가깝다.  
지금 필요한 것은 VC 투자 유치 자체보다 **문제 검증, MVP 데모, 베타 사용자 10~100명, 반복 사용 지표**를 만드는 것이다.

## 단계별 요약

| 단계 | 보통 상태 | 투자자 핵심 질문 | 미리 기준 |
| --- | --- | --- | --- |
| 0. 지원금/부트스트랩 | 아이디어, 문제정의, 초기 기획 | 왜 이 문제를 지금 이 팀이 풀어야 하나? | 현재 가장 가까움 |
| 1. 프리시드 | MVP/프로토타입, 초기 인터뷰, 파일럿 | 진짜 문제인가? 초기 사용자가 반응하는가? | 5월 MVP와 베타 10명 필요 |
| 2. 시드 | MVP 출시, 초기 사용자/매출, 반복 사용 | 시장 검증과 초기 PMF 신호가 있는가? | 베타 100명, 리텐션/전환 필요 |
| 3. 프리A | 초기 매출, 성장률, 리텐션 측정 | 시리즈A로 갈 만큼 성장 엔진이 보이는가? | 유료 전환과 반복 채널 필요 |
| 4. 시리즈A | PMF, 반복 가능한 GTM, 단위경제성 | 돈을 넣으면 확장되는 모델인가? | 아직 먼 단계 |
| 5. 시리즈B 이후 | 검증된 사업 확장 | 시장 지배력과 효율적 확장이 가능한가? | 참고만 |

## 0. 지원금·부트스트랩 단계

### 상태

- 법인/사업자 등록 전이거나 막 시작한 상태.
- 문제정의와 MVP 범위를 잡는 단계.
- VC 투자보다 정부지원, 액셀러레이터, 내부 프로그램, 파일럿 기회가 더 현실적.

### 평가 기준

| 평가축 | 보는 것 |
| --- | --- |
| 문제인식 | 고객 문제가 실제로 존재하는가 |
| 실현가능성 | 협약기간 또는 단기간 안에 MVP를 만들 수 있는가 |
| 성장전략 | 시장진입, 수익모델, 자금조달 계획이 있는가 |
| 팀 구성 | 대표자와 팀원이 문제를 풀 역량을 갖췄는가 |

### 미리에게 필요한 증거

- 중3~고3 학부모/학생 문제정의 1장.
- 미래 생기부 미리보기와 학교생활 로드맵 데모.
- 베타 인터뷰 10명 계획.
- 개인정보 처리 최소 기준.
- 5월 30일 기획심의용 MVP 범위.

## 1. 프리시드

### 상태

- MVP 또는 작동하는 프로토타입이 있음.
- 초기 고객 인터뷰, 파일럿, 대기자, 테스트 사용자가 있음.
- 매출은 없거나 매우 작아도 됨.
- 팀과 문제의 적합성, 초기 수요 증거가 핵심.

### 평가 기준

| 평가축 | 투자자가 보는 질문 | 미리에서 보여줄 자료 |
| --- | --- | --- |
| Founder-Market Fit | 왜 팀 310이 이 문제를 잘 풀 수 있는가? | 입시/AI/데이터/청소년 문제 경험 |
| 문제 강도 | 고객이 정말 돈·시간을 쓸 문제인가? | 학부모 인터뷰, 학생 인터뷰, 문제점 리스트 |
| MVP | 말이 아니라 작동하는 것이 있는가? | 미래 생기부 미리보기, 학교생활 로드맵, 하루/주간/월간/학기 액션 생성 데모 |
| 초기 반응 | 누가 써봤고 어떤 반응을 보였는가? | 베타 사용자 10명, 공유 의향, 재사용 의향 |
| 학습 속도 | 피드백을 얼마나 빨리 반영하는가? | 인터뷰 → 제품 수정 로그 |
| 시장 진입점 | 처음 먹을 좁은 시장이 명확한가? | 중3~고3 중 가장 강한 결제 타이밍과 ICP |

## 2. 시드

### 상태

- 제품이 공개 또는 제한 베타로 사용 중.
- 초기 사용자, 반복 사용, 일부 유료 전환 또는 강한 사용 의향이 있음.
- 문제-해결 적합성에서 제품-시장 적합성 초기 신호로 넘어가는 단계.

### 평가 기준

| 평가축 | 투자자가 보는 질문 | 미리에서 필요한 지표 |
| --- | --- | --- |
| 사용자 수요 | 사용자가 스스로 찾아오는가? | 베타 100명, 대기자, 추천 유입 |
| 반복 사용 | 한 번 보고 끝나지 않는가? | 2주/4주 재방문율, 리포트 갱신율 |
| 전환 | 무료에서 유료로 넘어갈 이유가 있는가? | 결제 의향, 사전예약, 유료 베타 |
| 리텐션 | 계속 쓸 이유가 있는가? | 월간 활동 기록률, 액션 완료율 |
| 채널 | 어떤 유입 채널이 작동하는가? | 맘카페, 학교 방문, 레퍼럴별 전환율 |
| 차별화 | 범용 AI 대비 왜 미리인가? | 누적 학생 데이터, 입시 맥락, 아카이빙 |

## 3. 프리A

### 상태

- 한국에서 시드와 시리즈A 사이의 브릿지 단계로 자주 쓰임.
- 초기 매출 또는 반복 사용이 확인되었지만 시리즈A 기준에는 아직 부족한 단계.
- 제품 완성도, 초기 수익모델, 성장률, 리텐션이 중요해짐.

### 평가 기준

| 평가축 | 투자자가 보는 질문 | 미리에서 필요한 지표 |
| --- | --- | --- |
| 초기 매출 | 누가 실제로 돈을 내는가? | 월 구독/학기 패키지 매출 |
| 성장률 | 사용자가 빠르게 늘고 있는가? | 월간 사용자/매출 성장률 |
| 리텐션 | 학기 단위로 유지되는가? | 월간 리포트 확인율, 재구독률 |
| BM 검증 | 가격과 제공 가치가 맞는가? | 월 3만 원 vs 학기 패키지 실험 |
| 운영 효율 | 고객 100명→1,000명으로 늘릴 수 있는가? | 자동 리포트 생성률, CS 비용 |
| 다음 라운드 가능성 | 시리즈A 지표로 이어질 수 있는가? | 반복 채널, LTV/CAC 가설 |

## 4. 시리즈A

### 상태

- PMF를 어느 정도 달성한 뒤, 이미 작동하는 성장 모델에 자금을 넣어 확장하는 단계.
- 투자자는 더 이상 가능성만 보지 않고 성장률, 리텐션, 단위경제성, 조직 확장성을 본다.

### 평가 기준

| 평가축 | 투자자가 보는 질문 | 미리에서 필요한 지표 |
| --- | --- | --- |
| PMF | 명확한 핵심 세그먼트에서 강한 유지율이 있는가? | 학기/연 단위 유지율 |
| 성장 모델 | 돈을 쓰면 고객이 반복적으로 늘어나는가? | CAC, 전환율, 채널별 ROAS |
| 수익성 구조 | LTV가 CAC보다 충분히 큰가? | LTV/CAC, gross margin |
| 제품 해자 | 경쟁사가 쉽게 따라 하기 어려운가? | 학생 데이터 그래프, 모범 데이터셋 |
| 조직 | 영업, 제품, CS, 법무를 확장할 수 있는가? | 팀 리더십, 운영 프로세스 |
| 시장 규모 | VC가 원하는 큰 결과가 가능한가? | TAM/SAM/SOM, 확장 시장 |

## 5. 시리즈B 이후

### 상태

- 이미 검증된 사업을 확장하는 단계.
- 국내 시장 점유율, 해외 진출, 신규 제품 라인, M&A, IPO 가능성이 중요.

### 미리 관점

- 지금 당장 준비할 단계는 아니다.
- 다만 장기 비전에는 B2G, 해외 한국 입시 준비생, 생애주기형 기록 플랫폼 확장을 옵션으로 남길 수 있다.

| 시장 | 문제 강도 | ICP 명확성 | 시장 진입 채널 | 큰 시장과 확장성 |

## 미리의 현재 단계 판단

### 현재 위치

**지원금/부트스트랩 단계 → 프리시드 준비 단계**

### 이유

- 문제정의와 타겟은 정리 중이지만 아직 반복 사용 지표가 없다.
- 미래 생기부 미리보기, 학교생활 로드맵, 하루/주간/월간/학기 액션 데모가 MVP 핵심이다.
- 베타 사용자 10명 확보와 실제 입력/재방문 데이터가 필요하다.
- 현재는 VC보다 정부지원, SOMA 내부 자원, 액셀러레이터 멘토링이 더 적합하다.

### 다음 단계 목표

| 기간 | 목표 | 지표 |
| --- | --- | --- |
| 5월 | 프리시드 준비 | 데모, 베타 10명, 인터뷰 인사이트 |
| 6월 | 프리시드 후보 | 베타 30~50명, 결과 공유율, 액션 선택률 |
| 7~8월 | 시드 준비 여부 판단 | 베타 100명, 4주 재방문율, 결제 의향 |

## 미리 기준 투자자에게 보여줄 자료

### 프리시드용

- 문제정의 1장
- 페르소나 1명
- 미래 생기부 미리보기와 학교생활 로드맵 데모
- 베타 사용자 반응
- 개인정보 처리 원칙
- 6개월 로드맵

### 시드용

- 사용자 세그먼트별 지표
- 4주 이상 사용 데이터
- 유료 전환 실험 결과
- 채널별 CAC 후보
- 리텐션/재방문 데이터
- 제품 자동화 구조

### 시리즈A용

- 반복 매출
- PMF 근거
- CAC/LTV
- 시장 확장성
- 팀 확장 계획
- 법무/운영 리스크 관리 체계
