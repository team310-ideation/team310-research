# 마케팅 카피 가이드

## 결론

MIRI의 마케팅은 `합격`, `세특 완성`, `AI 대필`처럼 보이는 표현을 피하고, **수행평가를 스스로 설계하는 과정 중심 도구**라는 점을 강조해야 한다.

## 금지 문구와 대체 문구

| 금지 문구 | 대체 문구 |
|---|---|
| AI가 대신 써주는 완벽한 세특 | 수행평가를 논리적으로 설계하는 탐구 플래너 |
| 클릭 한 번에 세특 완성 | 질문, 개요, TODO로 스스로 탐구를 시작 |
| 명문대 합격 보장 | 교과 기반 탐구 역량을 차근차근 쌓는 코치 |
| 선행 세특 완성 | 현재 배우는 교과 진도에 맞춘 심화 탐구 |
| 상위 1% 컨설턴트의 비밀 기재법 | 평가 기준에 맞춰 탐구 과정을 점검하는 체크리스트 |
| 이 앱 없으면 수시 실패 | 막막한 수행평가를 시작하기 쉽게 쪼개는 도구 |

## 권장 핵심 문장

- `막막한 수행평가, 질문부터 개요까지 스스로 설계하게 돕습니다.`
- `AI가 대신 쓰지 않습니다. 학생이 직접 탐구하도록 길을 나눕니다.`
- `교과 개념에서 출발해 탐구 질문과 다음 행동으로 이어집니다.`
- `완성 문단이 아니라 생각을 여는 질문과 체크리스트를 제공합니다.`
- `학부모에게는 결과보다 과정과 실행 흐름을 보여줍니다.`

## 앱 내 고지 문구

> MIRI의 AI 결과는 학생의 자기주도 탐구를 돕는 참고 자료입니다. 보고서, 발표문, 학교생활기록부 문장을 대신 작성하지 않으며, 최종 산출물은 학생 본인이 직접 작성해야 합니다.

## 랜딩 페이지 구성

1. 수행평가 입력
2. 오늘 할 일 1개 생성
3. 탐구 질문 선택
4. 개요와 자료 조사 키워드 확인
5. 완료 체크
6. 다음 꼬리 질문 추천
7. 월말 성장 리캡 공유

## 피해야 할 톤

- 학부모 불안을 자극하는 톤
- 입시 비밀 정보를 파는 톤
- 세특을 조작하거나 꾸며주는 톤
- 학교/교사의 역할을 대체하는 톤
