# 서비스 작동구조

## 작동 사이클

1. 온보딩: 기존 생기부·활동 자료와 AI 인터뷰 답변 입력
2. 초기 분석: 학과·진로 적합도, 강점, 약점, 현재 데이터 상태 분석
3. 학교생활 플랜 생성: 과목, 동아리, 도서, 대회, 수행평가, 탐구, 세부능력특기사항 방향 추천
4. 실행 후 아카이빙: 수행평가, 레포트, 발표 자료, 활동 결과물을 누적
5. 다음 액션 추천: 이전 데이터 맥락 기반 하루/주간/월간/학기 액션 제시
6. 갱신: 활동 데이터가 쌓일수록 미래 생기부와 학교생활 로드맵 업데이트

## 데이터 흐름

```text
기존 생기부·활동 자료(사용자 데이터) + 모범 생기부(참조 데이터)
      ↓
AI 생성·분석
      ↓
중3~고3 학교생활 의사결정 액션 아이템
      ↓
사용자 실행 + 데이터 누적
      ↓
업데이트된 생기부 및 다음 액션
```

## 핵심 기능 범위

### Must Have

- 기존 생기부·활동 자료 입력
- AI 인터뷰 기반 학생 맥락 수집
- 미래 생기부 미리보기 생성
- 중3~고3 과목·활동·탐구 선택 액션 제시
- 사용자 활동 데이터 아카이빙

### Later

- 모범 생기부 RAG 데이터셋 고도화
- 프롬프트 교육 및 스킬 공유 커뮤니티
- 멘토 매칭 또는 전문가 연결

## 반복 사용 구조

| 반복 지점 | 사용자 가치 |
| --- | --- |
| 하루 | 오늘 해야 할 작은 액션 확인 |
| 주간 | 수행평가·활동·탐구 진행 상황 점검 |
| 월간 | 활동 공백, 진로 일관성, 다음 선택 점검 |
| 학기 | 생기부 방향성, 과목·활동 선택, 로드맵 갱신 |
| 학년 | 목표 학과·전형 기준으로 학교생활 서사 재정렬 |

## 차별화 구조

- 단순 "분석 후 추천"이 아니라 고3까지 이어지는 학교생활 의사결정 관리에 초점을 둔다.
- 핵심 차별화는 학생 데이터 아카이빙, 입시 맥락 유지, 액션 갱신에 있다.
- 모범 생기부 데이터셋은 범용 AI 대비 장기 데이터 우위 후보로 둔다.
